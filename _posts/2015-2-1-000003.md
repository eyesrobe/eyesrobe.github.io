------------------------------------
スクリプト

```

wiringPi2test_with_sudo.py

import time
import wiringpi2

wiringpi2.wiringPiSetupGpio()
wiringpi2.pinMode(18,1)
wiringpi2.digitalWrite(18,1)
time.sleep(2)
wiringpi2.digitalWrite(18,0)

```
